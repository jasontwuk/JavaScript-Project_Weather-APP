# JavaScript Project_Weather App

<p>This is a weather app that shows today's and the future week's weather. London is the default city, you can change it to any city you like by clicking the edit button on the right. It will show your latest chosen city when you reopen the app next time.</p>

<h4>Used technologies</h4>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>Vanilla JS</li>
</ul>

<h4>Used APIs</h4>
<ul>
  <li>GoogleMap</li>
  <li>OpenWeatherMap </li>
</ul>
